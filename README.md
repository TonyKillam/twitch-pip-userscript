# Twitch Custom PiP Button (Stable Layout)

A high-performance Tampermonkey userscript that adds a native, stable Picture-in-Picture (PiP) button to the Twitch.tv video player interface. 

## 🚀 Features

* **Instant Load**: Uses a fast `MutationObserver` instead of heavy background loops. It loads instantly without lagging your browser.
* **Stable Layout**: Anchored to the far right side of the control panel. It will not move or jump when the source quality tag drops in or out.
* **Keyboard Shortcut**: Press the **`P`** key on your keyboard to toggle PiP mode instantly. (Automatically disables when you are typing in the chat box!)
* **Smart Memory**: Remembers your last floating window size and screen placement automatically.

## 🛠️ Installation

1. Make sure you have the [Tampermonkey Extension](https://tampermonkey.net) installed in your browser.
2. Click the link below to open the raw code file:
   * **[👉 Click Here to Install Directly](PASTE_YOUR_RAW_GITHUB_URL_HERE)**
3. Tampermonkey will automatically detect the script. Click **Install**.

## ⌨️ How to Use

* **Click**: Click the rectangular screen icon in the bottom right corner of the Twitch video player.
* **Hotkey**: Tap **`P`** on your keyboard while watching any stream.
* **Resizing**: Drag the corners of the window to change its size. Your browser will remember this exact size for next time!
